## 9.1 스케일링
### 9.1.1 스케일링이란
- IT 자원을 확장하거나 축소하는 기능
- 고가용성과 내결함성을 확보
### 9.1.2 스케일링의 종류
- 수직 스케일링
	- 자원의 용량을 확장하거나 축소하는 것
	- 스케일 업, 스케일 다운
- 수평 스케일링
	- 자원의 수량을 확장하거나 축소하는 것
	- 스케일 인, 스케일 아웃
## 9.2 AWS 오토 스케일링 서비스
- 스케일링으로 클라우드 자원을 '동적으로' 확장하거나 축소하여 자원의 규모를 조정
### 9.2.1 Amazon EC2 오토 스케일링이란
- 동적으로 EC2 인스턴스 수를 확장하거나 축소하여 워크로드를 유지하는 서비스
- 오토 스케일링 그룹으로 모음을 생성
### 9.2.2 AWS EC2 오토 스케일링 구성 요소
- 그룹
- 구성 템플릿
- 조정 옵션
### 9.2.3 Amazon EC2 오토 스케일링의 인스턴스 수명 주기
- 인스턴스 확장에 따른 이벤트
	- 대기 중 -> 실행 중
- 인스턴스 축소에 따른 이벤트
	- 종류 중 -> 종료됨
### 9.2.4 Amazon EC2 오토 스케일링 조정 옵션
- 인스턴스를 잉정한 수로 유지
- 수동 조정
- 동적 조정
	- 대상 추적 조정
	- 단계 조정
	- 단순 조정
- 일정을 기반으로 예약된 조정
- 일정을 기반으로 예측 조정
### 9.2.5 Amazon EC2 오토 스케일링 요금
- 별도 발생 요금은 없고 EC2 인스턴스와 CloudWatch 경보에 따른 요금만 발생
